# SRT to Premiere Pro JSON

A lightweight Python utility designed to convert SRT subtitle files—specifically those generated by Whisper (tested with the Large-v3-turbo model using the Buzz app)—into a JSON transcript format that Adobe Premiere Pro can accurately interpret.

## Purpose
This script bridges the gap between AI transcription (like OpenAI's Whisper) and Adobe Premiere Pro. It transforms standard `.srt` subtitle blocks into Premiere-friendly JSON structures, making transcriptions immediately usable for further editing without manual reformatting.

## Features
- **Accurate Parsing**: Converts block-level `.srt` files into Premiere's granular word-and-segment-based JSON format.
- **Dependency-Free**: Uses only Python 3 standard libraries (`json`, `re`, `uuid`, `os`, `sys`).
- **macOS Friendly**: Includes a built-in patch for terminal drag-and-drop that fixes missing tildes in iCloud paths (`comappleCloudDocs` → `com~apple~CloudDocs`).

## Usage
Run the script from your terminal, passing the path to the `.srt` file as an argument:

```bash
python srt_to_json.py /path/to/your/subtitle.srt
```

### Example

```bash
python srt_to_json.py ~/Desktop/interview_transcript.srt
```

This will automatically generate a JSON transcript in the exact same directory, using the original base file name: `~/Desktop/interview_transcript.json`. You can then import this directly into Adobe Premiere Pro.
